# Ozora Festival Companion v3.0

Upload **all files in this folder** to the root of the GitHub repository. This flat build intentionally uses no subfolders and matches GitHub's web upload layout shown in your repository.

GitHub: **Settings → Pages → Deploy from a branch → main → /(root)**.

The schedule currently contains representative sample data because the original uploaded source files were not available in this workspace. Replace `DATA.schedule` and `DATA.artists` at the top of `app.js` with the final festival data.

Browser reminders are best-effort while the app is open. Reliable alarms after the browser is fully closed require a push service or native app.
