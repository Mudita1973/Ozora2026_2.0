# Ozora Festival Companion v3.0

Upload **the contents of this folder** to the root of the GitHub repository. `index.html` must be at repository root, not inside another folder.

GitHub: **Settings → Pages → Deploy from a branch → main → /(root)**.

The schedule currently contains representative sample data because the original uploaded source files were not available in this workspace. Replace `DATA.schedule` and `DATA.artists` at the top of `assets/app.js` with the final festival data.

Browser reminders are best-effort while the app is open. Reliable alarms after the browser is fully closed require a push service or native app.
